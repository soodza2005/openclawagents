## 📦 Frameworks & Libraries

### React (JavaScript UI Library)

| Version | Release Date | Status |
|---------|-------------|--------|
| React 19 | ธันวาคม 2024 | ✅ Stable |
| React 19.2 | ตุลาคม 2025 | ✅ Stable |
| React 19.3 | อนาคต | 🔄 Coming |

#### 🔥 React 19 Features ที่ควรรู้

- **React Compiler v1.0** (ตุลาคม 2025) — Auto-optimize renders
- **Server Components** — Render บน server ได้เลย
- **Actions** — Handle form submissions แบบใหม่
- **useEffectEvent** — แยก logic ออกจาก effects
- **Activity** — Streaming SSR ดีขึ้น

#### ⚠️ React Important Notices

- **Deprecated: Create React App** (กุมภาพ 2025) — แนะนำย้ายไป Vite/Next.js
- **Security Alert:** Critical RCE vulnerability in React Server Components (ธันวาคม 2025) — อัปเกรดทันที!
