# Python Tools Knowledge - 2026-03-06

## แหล่งข้อมูล: Awesome Python (github.com/vinta/awesome-python)

### 🛠️ Tools ที่เกี่ยวข้องกับงาน HRMS/Project Management

#### 1. Admin Panels (สำหรับสร้าง Dashboard สำหรับ HR)
- **Streamlit** - สร้าง dashboards, reports, chat apps ได้ง่ายมาก
- **django-grappelli** / **django-unfold** - Admin interface สวยๆ สำหรับ Django
- **flask-admin** - Admin interface สำหรับ Flask
- **jet-bridge** - Admin panel framework สำหรับทุก application

#### 2. Data Analysis & Visualization
- **pandas** - Data manipulation อันดับ 1
- **numpy** - Numerical computing
- **matplotlib** / **seaborn** - Data visualization
- **plotly** - Interactive charts

#### 3. Database & ORM
- **SQLAlchemy** - ORM ยอดนิยม
- **Django ORM** - สำหรับ Django apps
- **psycopg2** - PostgreSQL driver
- **pymysql** - MySQL driver

#### 4. Web Frameworks
- **Django** - Full-stack framework (เหมาะกับ HRMS)
- **FastAPI** - Modern, fast API framework
- **Flask** - Lightweight framework
- **FastAPI** - รองรับ async, auto documentation

#### 5. Authentication & Security
- **Authlib** - OAuth/OpenID implementation
- **python-jose** - JWT handling
- **bcrypt** - Password hashing
- **pydantic** - Data validation

#### 6. API & HTTP
- **requests** - HTTP client ยอดนิยม
- **httpx** - Async HTTP client
- **fastapi** - Built-in API with Swagger docs

#### 7. Task Queue & Scheduling
- **Celery** - Distributed task queue
- **APScheduler** - Job scheduling
- **python-cron** - Cron-like scheduling

#### 8. Testing
- **pytest** - Testing framework
- **pytest-django** - Django testing
- **coverage** - Code coverage

#### 9. Date & Time
- **python-dateutil** - Date utilities
- **pytz** - Timezone handling

#### 10. Data Validation
- **pydantic** - Data validation using Python type hints
- ** Cerberus** - Lightweight data validation
