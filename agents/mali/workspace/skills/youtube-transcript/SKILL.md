# Skill: YouTube Transcript

## Summary
ดึง transcript/subtitle จาก YouTube video

## Usage
```
@transcript <youtube_url>
```
หรือ
```
ดึง transcript จาก <youtube_url>
```

## Example
```
@transcript https://www.youtube.com/watch?v=WqWMszBB9t0
```

## Notes
- รองรับทั้ง URL แบบเต็มและแบบย่อ
- จะดึง subtitle ภาษาอังกฤษเป็นค่าเริ่มต้น
- บันทึกไฟล์ที่ /tmp/video*.vtt
