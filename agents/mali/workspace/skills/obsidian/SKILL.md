# Obsidian Skill

จัดการ Obsidian notes ผ่าน command line

## การใช้งาน

```bash
# สร้าง note
obsidian new "ชื่อ note"

# ค้นหา
obsidian search "คำค้น"

# เปิด vault
obsidian open "vault-name"
```

## ติดตั้ง

```bash
brew install obsidian
```

## Note

ต้องมี Obsidian installed และ vault ที่ต้องการ
